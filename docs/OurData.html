About our Data

We were lucky to stumble on Lahman Database, a large record of historical baseball data. There were many different files available to chose from, which was our first issue, figuring out what data we actually needed and what would be useful to us. After figuring this out, we were able to use Python to merge the data we needed to create a final DataFrame as seen below. This large dataset contained over 3000 unique players, which only a small percentage were eventually inducted. Rather than focusing on individual positions, we wanted to instead take a larger dataset to be able to compare player statistics overall vs if they were inducted instead of positional based comparisons. All credit for our data goes to Sean Lahman.

Questions We Hope to Answer

- Why has the number of players inducted rapidly decreased?
- What differentiates a player that gets inducted and one that does not?
- What key metrics/statistics have a strong correlation with induction?
- Is there any way to accurately guess whether a player is inducted?
- Can Machine Learning be used to give us an accurate guess on player induction based on statistics?
