How We Used Machine Learning

First we took our two large datasets "People.csv", which provides us basic player info such as birth date, weight, and height as well as our second data set "HallofFame.csv" which had our votes needed, votes received, and whether they were inducted or not into the Hall of Fame. We turned both of these into DataFrames that we would later merge into one to have all of our data in one set. The next obstacle was to create and clean our full dataset which would contain our career batting, fielding, and pitching data which we will use to train our Machine Learning Algorithm. We used classification to predict the inducted status based on all the stats within our full dataset. We used logistic regression to see how well we could accurately predict whether a player is inducted or not based on their game statistics within their career.
